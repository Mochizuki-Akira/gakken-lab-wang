電池と電気分解

十一、電池と電気分解

0. イオン化傾向

Li > K > Ca > Na > Mg > Al > Zn > Fe > Ni > Sn > Pb > H2 > Cu > Hg > Ag > Pt > Au

1. 電池の基礎

1-1. 電池とは

電池：酸化還元反応を利用して、化学エネルギーを電気エネルギーとして取り出す装置。

1-2. 電池の基本構造と電流の向き

基本の考え方

  • 電子は負極（酸化が起こる電極）から外部回路へ出て、正極（還元が起こる電極）へ流れ込む。
  • 溶液中では、電気的中性を保つために陽イオン/陰イオンが移動する（外部回路は電子、溶液はイオン）。
  • 電池の記述は「負極の酸化」と「正極の還元」の半反応式を立て、電子数をそろえて加える。

1-3. 分極と減極

分極：正極（多くはCuなど）表面に水素などが付着し、電子の受け渡しが進みにくくなることで、電流が弱くなる現象。

減極剤：分極の原因物質（例：H2）を消費する/発生させないために用いる物質（例：MnO2など）。

2. 一次電池（ボルタ電池・ダニエル電池）

2-1. ボルタ電池

代表例：Zn と Cu を希硫酸などに入れて作る簡単な電池。起電力は得られるが分極が起こりやすい。

半反応式（典型例）

  • 負極（酸化）：Zn → Zn2+ + 2e−
  • 正極（還元）：2H+ + 2e− → H2
  • 全反応：Zn + 2H+ → Zn2+ + H2

正極表面にH2が付着して分極しやすい（電流が弱くなる）。

2-2. ダニエル電池

ZnSO4水溶液中のZn板（負極）と、CuSO4水溶液中のCu板（正極）を、隔膜や塩橋でつないだ電池。分極が起こりにくく、安定な電流を取り出せる。

半反応式

  • 負極（酸化）：Zn → Zn2+ + 2e−
  • 正極（還元）：Cu2+ + 2e− → Cu
  • 全反応：Zn + Cu2+ → Zn2+ + Cu

ポイント：

  • Zn側ではZnが溶けてZn2+が増える。
  • Cu側ではCu2+が減ってCuが析出する。
  • 溶液の電気的中性を保つため、塩橋中のイオンが移動する（例：陰イオンはZn側へ、陽イオンはCu側へ）。

3. 二次電池（鉛蓄電池）

3-1. 鉛蓄電池の特徴

二次電池：放電後に外部電源で逆向きに電流を流し、元の物質に戻して再利用できる電池。

鉛蓄電池は自動車のバッテリーとして代表的。電解液は希硫酸（H2SO4）。

3-2. 放電時の反応

放電（電池として働く向き）

  • 負極（Pbの酸化）：Pb + SO42− → PbSO4 + 2e−
  • 正極（PbO2の還元）：PbO2 + 4H+ + SO42− + 2e− → PbSO4 + 2H2O
  • 全反応：Pb + PbO2 + 2H2SO4 → 2PbSO4 + 2H2O

放電が進むと、両極にPbSO4が生成し、電解液中のH2SO4が減るため濃度（比重）が下がる（状態判定に利用）。

3-3. 充電時の反応

外部電源で逆向きに電流を流すと、放電反応が逆向きに進み、PbとPbO2が再生する。

  • 「放電の全反応」を逆に書いたものが充電の全反応。
  • 充電ではH2SO4が増え、比重が上がる。

4. 燃料電池

4-1. 燃料電池の考え方

燃料電池：燃料（代表：H2）と酸化剤（代表：O2）を外部から連続供給し、反応により電気を取り出す電池。燃料が供給される限り発電できる。

4-2. 水素-酸素燃料電池（代表例）

典型的な半反応式（酸性電解質の例）

  • 負極（酸化）：H2 → 2H+ + 2e−
  • 正極（還元）：O2 + 4H+ + 4e− → 2H2O
  • 全反応：2H2 + O2 → 2H2O

生成物が水で、排出物が少ないことから環境負荷が小さい（ただし燃料製造・貯蔵など課題もある）。

5. 電気分解

5-1. 電気分解とは

電気分解：外部電源で電流を流し、電極で酸化還元反応を起こして物質を生成する操作。

  • 陰極（−）：電子が供給される → 還元が起こる。
  • 陽極（＋）：電子が奪われる → 酸化が起こる。

5-2. 電極で起こる反応の優先順位

① 陽極（酸化：電子を失う）

  1. 電極が Pt、C 以外の金属か？
    YES → 金属が溶ける（電極が溶解）
    例：Cu → Cu2+ + 2e−
  2. NO の場合：電解液にハロゲン化物イオン（Cl−、Br−、I−）を含むか？
    YES → ハロゲン単体が発生（Cl2、Br2、I2 など）
  3. NO の場合：O2 が発生（主に水/水酸化物イオンの酸化）
    ・液性が中性〜酸性：2H2O → O2 + 4H+ + 4e−
    ・液性が塩基性：4OH− → O2 + 2H2O + 4e−

② 陰極（還元：電子を受け取る）

  1. 電解液に Cu2+ や Ag+ を含むか？
    YES → Cu や Ag が析出（イオンが還元され金属になる）
    Cu2+ + 2e− → Cu
    Ag+ + e− → Ag
  2. NO の場合：H2 が発生
    ・液性が酸性：2H+ + 2e− → H2
    ・液性が中性〜塩基性：2H2O + 2e− → H2 + 2OH−

覚え方（簡単）

  • 陽極：「金属が溶ける → ハロゲン → 水（O2）」
  • 陰極：「金属イオン（Cu2+, Ag+）→ 水/水素イオン（H2）」

5-3. 代表例

(1) CuCl2水溶液（不活性電極）

  • 陰極：Cu2+ + 2e− → Cu（銅の析出）
  • 陽極：2Cl− → Cl2 + 2e−（塩素発生）

(2) NaCl水溶液（食塩水、濃い場合の代表）

  • 陰極：2H2O + 2e− → H2 + 2OH−（水素発生）
  • 陽極：2Cl− → Cl2 + 2e−（塩素発生）
  • 溶液：NaOHが生成（OH−増加）

(3) 希硫酸（＝水の電気分解）

  • 陰極：2H+ + 2e− → H2
  • 陽極：2H2O → O2 + 4H+ + 4e−
  • 全反応：2H2O → 2H2 + O2

(4) CuSO4水溶液

電極が白金（不活性）か銅（活性）かで陽極反応が変わる。

  • 不活性電極：陰極でCu析出、陽極でO2発生（硫酸イオンは基本的に放電しにくい）。
  • 銅電極：陰極でCu析出、陽極はCu → Cu2+ + 2e−（銅が溶ける）。溶液中のCu2+濃度がほぼ一定になりやすい。

6. 電気量と生成量（ファラデーの法則）

6-1. 電気量Q

Q = It（Q：C、I：A、t：s）

6-2. 電子の物質量

n(e−) = Q / F

ファラデー定数：F = 9.65×104 C/mol

6-3. 生成物の物質量・質量・体積

  • 半反応式の電子数を見て、n(生成物)を求める。
  • 質量：m = nM
  • 気体体積（標準状態）：V = 22.4 n（L）

6-4. 典型問題の解き方（型）

  1. 電流Iと時間tからQを出す（単位注意：分→秒）。
  2. Q/Fで電子のmol数を出す。
  3. 半反応式の係数比から生成物のmol数へ変換する。
  4. 必要なら質量・体積へ。
